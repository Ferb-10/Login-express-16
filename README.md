# Login-express-16

## Overview
This project is a simple Node.js web application that demonstrates user authentication using Express middleware.  
Users submit a username and password through a form, and access to a protected page is granted only if the credentials are correct.

![image alt](https://github.com/Ferb-10/Login-express-16/blob/61e26ba443396443c0c75e3d4ce6f7252b007bd4/public/img/login-project.png)

## Features
- Form-based user authentication  
- Custom middleware for password validation  
- Conditional routing based on authentication status  
- Serving static files using Express  
- Simple access control to a protected page  

## Technologies Used
- Node.js  
- Express  
- Body-parser  
- HTML/CSS  

## What I Learned
- How to build and use custom middleware in Express  
- Handling form data using body-parser  
- Implementing basic authentication logic  
- Managing request-response flow in a server environment  
- Serving static files and HTML pages with Express

## Overview
本プロジェクトは、Expressのミドルウェアを用いたシンプルな認証機能を実装したNode.jsアプリケーションです。  
ユーザーがフォームに入力したユーザー名とパスワードをもとに認証を行い、正しい場合のみ保護されたページにアクセスできます。

## Features
- フォーム入力によるユーザー認証  
- カスタムミドルウェアによるパスワード検証  
- 認証結果に応じたページ遷移  
- Expressによる静的ファイル配信  
- シンプルなアクセス制御機能  

## Technologies Used
- Node.js  
- Express  
- body-parser  
- HTML / CSS  

## What I Learned
- Expressにおけるミドルウェアの仕組みと実装方法
